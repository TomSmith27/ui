## Usage

Use a `v-model` to make the Input reactive.

::component-example
#default
:slider-example

#code

```vue
<script setup>
const value = ref("");
</script>

<template>
  <USlider v-model="value" />
</template>
```

::

### Style

Use the `color` prop to change the visual style of the Slider.

## ::component-card

baseProps:
name: 'slider'
placeholder: 'Search...'
props:
color: 'lime'

---

::

### Size

Use the `size` prop to change the size of the Input.

## ::component-card

baseProps:
name: 'slider'
props:
size: 'sm'

---

::

### Disabled

Use the `disabled` prop to disable the Input.

## ::component-card

baseProps:
name: 'input'
placeholder: 'Search...'
props:
disabled: true

---

::
